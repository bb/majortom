= MaJorToM Documentation =

The MaJorToM ( Merging Topic Maps Engine ) project was founded to develop a lightweight, merging and flexible Topic Maps engine satisfying different business use cases. The engine provides a couple of new features above to other engines based on the http://www.tmapi.org/2.0/[Topic Maps API version 2.0]. 

The project is hosted on http://code.google.com/p/majortom/. If you find bugs or have feature requests please file a http://code.google.com/p/majortom/issues/list[ticket].

The JavaDocs of the MaJorToM API are located http://docs.topicmapslab.de/majortom/javadoc/index.html[here].
